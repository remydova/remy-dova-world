# remy-dova-world

A quiet digital space for introverts and quiet souls — Remy's world.

### What is this?
- Cozy, handcrafted hub with Remy's lo-fi aesthetic
- Zine builder for gentle, lowercase reflections and affirmations
- Links to the Lofi Hideout app[](https://remydova.github.io/lofi-hideout/)

### Core vibe
- lowercase, breathy, reassuring
- rainy Midwest windows, string lights, plants, crochet
- teal/amber/lavender on deep navy
- it's okay to be you. no guilt. no fixes.

### Current features
- zine-builder.html → generate simple themed zines (e.g., "introvert life")
- future plans: fan-build pages, guestbook integration, more quiet tools
